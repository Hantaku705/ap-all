## Webapp（コンセプト学習）

### 本番URL
https://webapp-five-bay.vercel.app

### 技術スタック
- Next.js 16.1.3 (App Router)
- React 19
- TypeScript
- Tailwind CSS

### ページ構成
| パス | 内容 |
|-----|------|
| `/` | トップページ |
| `/concept` | コンセプト学習トップ（2セクション） |
| `/concept/textbook` | コンセプトの教科書（全6章） |
| `/concept/examples` | 事例一覧（15件） |
| `/concept/examples/[slug]` | 事例詳細（14項目評価） |

### Key Files
| ファイル | 用途 |
|---------|------|
| `webapp/src/data/concept-data.json` | コンセプトデータ（※手動同期） |
| `webapp/src/lib/concept.ts` | データ取得関数・型定義 |
| `webapp/src/app/concept/textbook/page.tsx` | 教科書ページ（全6章） |
| `webapp/src/app/concept/examples/[slug]/page.tsx` | 事例詳細ページ |
| `docs/concept/concept-data.json` | マスターデータ（こちらを編集） |

### データ同期について
`docs/concept/concept-data.json` がマスター。編集後は `webapp/src/data/concept-data.json` にコピーが必要。
**理由**: Next.js 16 Turbopack がシンボリックリンクを解決できないため。

## コンセプト学習資料

### docs/concept/
| ファイル | 内容 |
|---------|------|
| `README.md` | 概要 |
| `concept-textbook.md` | コンセプトの教科書（全6章） |

### 教科書の構成（6章）
1. コンセプトの公式（リフレーミング公式）
2. 良いコンセプトの4要素
3. いい商品コンセプトの5条件
4. 良いコンセプトの7条件（UGC向け）
5. 定義の使い分け
6. 評価項目一覧（14項目）

### 事例一覧（15件）
1. 塗るつけまつげ（デジャヴュ）
2. コンビニジム（チョコザップ）
3. 夜間美容シャンプー（Yolu）
4. 顔面死守ファンデ（フィットミー）
5. 紫外線、浴びたら洗おう（ソラミー）
6. お、ねだん以上。（ニトリ）
7. コンビニコーヒー（セブンカフェ）
8. 飲む点滴（甘酒）
9. 1000円カット（QBハウス）
10. 大人のふりかけ（丸美屋）
11. 朝マック（マクドナルド）
12. 置くだけWi-Fi（SoftBank Air）
13. 落ちないリップ（リップモンスター）
14. 飲むスキンケア（ディフェンセラ）
15. ながら美顔器（ヤーマン）

### ネーミングパターン
| パターン | 例 | 特徴 |
|---------|----|----|
| 行動+結果 | 塗るつけまつげ | 何をすると何が得られるか |
| 既知概念+商品 | コンビニジム | 馴染みのある概念を転用 |
| 時間+商品 | 夜間美容シャンプー | タイミングで差別化 |
| 強調語+商品 | 顔面死守ファンデ | インパクトのある言葉 |
| 対象+商品 | 大人のふりかけ | ターゲットを明示 |
| 行動提案型 | 浴びたら洗おう | 新習慣を提案 |

### 評価項目（14項目）

**マスト（7項目）- コンセプトの本質**
1. 新しい視点・独自性があるか
2. 顧客インサイトに応えているか
3. ベネフィットが明確か
4. シンプルで伝わりやすいか
5. 行動・購買意欲を生むか
6. 信じられる理由があるか
7. 誤解なく伝わるか

**任意（7項目）- 拡散力強化**
8. 口に出したくなる・語呂が良いか
9. 感情を喚起するワードがあるか
10. 使った自分を演出できるか
11. ストーリー性・妄想できるか
12. 擬音・造語を活用しているか
13. ハッシュタグ化しやすいか
14. 主張の核心が込められているか
