## ゲーム構造（6種類）

### 1. 再生数ゲー（移動平均ゲー）

| 項目 | 内容 |
|------|------|
| 勝利条件 | 1投稿 80,000 views |
| 本質 | 移動平均ゲー（前の動画が次に影響） |
| メカニズム | 過去7-14本の再生数平均がアルゴリズムに影響 → 低迷すると次も低迷 |
| 勝ちパターン | 低迷時に企画動画を投下して移動平均を回復 |

#### 再生数の3変数

| 変数 | 現状 | 説明 |
|------|------|------|
| **アカウント質** | △ | アカウントの信頼性・アルゴリズム評価 |
| **投稿質** | △ | コンテンツのクオリティ・視聴維持率 |
| **投稿数** | ⚪︎ | 投稿頻度・量 |

#### アカウント質の改善

| 施策 | 内容 |
|------|------|
| **マニュアル** | 全員で1日30分動画を視聴（エンゲージメント向上） |
| **リサーチ&アカウント質工場** | 系統的なアカウント育成 |
| **スマホ農場** | Phone Farmによるアカウント質向上 |

**スマホ農場の参考資料**:
- やり方: https://note.com/taku_sid/n/n77875725bcf9
- まとめ: https://phonefarm-threat-intel.vercel.app/
- 購入品: https://some3c.com/products/10-usb-ports-phone-holder-farm-box-usb-charging-stand

#### 投稿質の改善

| 施策 | 内容 |
|------|------|
| **顔出し投稿** | モデル採用も考慮 |
| **新商品** | 積極的にギフティングをもらう ↔ 営業連携 |
| **検索上位** | 検索上位のコンテンツを真似る |

#### 投稿数の改善

| 施策 | 内容 |
|------|------|
| **毎日投稿** | 業務委託と連携してIGチームは構築済み |

### 2. TTSゲー（初動ハックゲー）

| 項目 | 内容 |
|------|------|
| 勝利条件 | GMV MAX乗せ → TikTok Shop売上 |
| 本質 | 初動ハックゲー（身内購入 → コメントハック） |
| メカニズム | 最初の20件購入でアルゴリズムブースト → コメントで商品FAQ化 |
| 勝ちパターン | 身内に購入依頼、コメントスクリプト用意、毎日コメント返信 |

### 3. 売上ゲー（ループゲー）

| 項目 | 内容 |
|------|------|
| 勝利条件 | 粗利 2,000〜5,000万円/月 |
| 本質 | ループゲー（ORG再生 → 問い合わせ → 案件 → 実績 → ORG再生） |
| メカニズム | 再生数が問い合わせを生み、案件獲得で実績になり、さらに再生を呼ぶ |
| 勝ちパターン | インバウンド経路の確立（検索上位、バズ動画） |

### 4. 広告配信ゲー（PKGゲー）

| 項目 | 内容 |
|------|------|
| 勝利条件 | マージン獲得（KB 4-5%） |
| 本質 | PKGゲー（素材納品 + 広告配信セット売り） |
| メカニズム | 素材だけでなく配信まで含めることで粗利率向上 |
| 勝ちパターン | 店舗訴求PKG（ウェルシア等小売タイアップ）、全媒体アンプリファイ |

### 5. ライブコマースゲー（箱ゲー）

| 項目 | 内容 |
|------|------|
| 勝利条件 | ライブでの購入促進 |
| 本質 | 箱ゲー（購買層を集める「箱」を作る） |
| メカニズム | 購買意欲の高い視聴者を集める「箱」（場）を作れば売れる |
| 勝ちパターン | Gracemode型（未経験×センスあり採用 → 研修 → ライブ配信） |

### 6. 大量動画ゲー（編集者ネットワークゲー）

| 項目 | 内容 |
|------|------|
| 勝利条件 | 月1,800件生成 |
| 本質 | 編集者ネットワークゲー |
| メカニズム | 演者×編集者×AIで大量動画生成、数で勝負 |
| 勝ちパターン | Senjin Holdings型（TTO、編集者ネットワーク構築） |
