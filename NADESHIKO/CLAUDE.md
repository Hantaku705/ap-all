AnyMind Groupの美容系SNSメディア「NADESHIKO」の運用資料・分析データを管理するプロジェクト。

---

## 事業概要

| 項目 | 内容 |
|------|------|
| 事業 | 美容系SNSメディア運営（TikTok/IG/YT/X） |
| 親会社 | AnyMind Group |
| ビジネスモデル | タイアップ投稿 + 広告配信（アンプリファイ）+ 素材納品 |

---

## 改善戦略

1. **バズ企画** - 移動平均を上げるため、企画でバズ動画を連発
2. **検索対策** - 検索上位コンテンツを真似る
3. **アカウント質向上** - 全員で1日30分動画視聴、スマホ農場検討
4. **投稿数増加** - 毎日投稿 + 業務委託チーム構築
5. **顔出し・新商品** - モデル採用、ギフティング積極獲得

## Webアプリ

### 売上管理ダッシュボード

| 項目 | 値 |
|------|-----|
| 本番URL | https://nadeshiko-sales.vercel.app |
| 技術スタック | Next.js 16.1.4 + React 19 + Recharts + Tailwind CSS |

**4タブ構成**:
| タブ | 内容 |
|-----|------|
| Dashboard | KPIサマリー、月次推移グラフ、AJP/RCP比率 |
| Deals | 案件一覧（編集・追加・削除機能） |
| Performance | 担当者別・アカウント別・クライアント別ランキング |
| Settings | 月別目標設定、CSVエクスポート |

**開発コマンド**:
```bash
cd NADESHIKO/webapp
npm run dev
```

**デプロイ**:
```bash
cd NADESHIKO/webapp
vercel --prod --yes
```
