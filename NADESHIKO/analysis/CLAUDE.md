## tiktok-decline-analysis.md

### 概要

| 項目 | 値 |
|------|-----|
| 分析期間 | 2025年10月〜2026年1月 |
| 分析対象 | TikTok投稿 756件 |
| 主要発見 | 平均再生数 -86%（286,341→39,821 views/post） |

### 主要発見

| 指標 | 10月 | 1月 | 変化 |
|------|------|-----|------|
| 平均再生数 | 286,341 | 39,821 | -86% |
| バズ率（10万+） | 30.8% | 7.5% | -23.3pt |
| フル視聴率 | 20.3% | 11.2% | -9.1pt |
| PR投稿数 | 39 | 5 | -87% |

### 最も下落したアカウント（Top 5）

| アカウント | 10月 | 1月 | 変化率 |
|-----------|------|-----|--------|
| kana | 627K | 13K | -97.9% |
| ビビちゃん | - | - | -96.9% |
| 肌コミちゃん | - | - | -96.1% |
| 突撃ちゃん | - | - | -95.8% |
| モテコスメちゃん | - | - | -94.5% |

### 主要原因

1. **11月第5週の大量投稿**（187件）→ 移動平均低下
2. **PR案件激減**（87件→5件、-94%）
3. **メガバズ消失**（100万+再生がほぼゼロ）
4. **フル視聴率低下**（20.3%→11.2%）
