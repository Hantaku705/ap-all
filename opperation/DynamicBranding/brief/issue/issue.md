## フェーズ1: 双方向性の把握（完了）

### Issue 1: ブランド間の双方向性

#### 問いの背景

味の素グループには複数のブランドが存在するが、各ブランドの施策は個別に展開されている。しかし実際には、あるブランドの成長が別のブランドに影響を与えている可能性がある。この「ブランド間の相関関係」を可視化できれば、施策の優先順位づけや投資配分の最適化が可能になる。

#### 具体的な問い

**Q1-1. 正の相関**
> あるブランドが伸びると、連動して伸びるブランドはあるか？

- ほんだし↑ → クックドゥ↑ のような関係性はあるか
- 同じCEP（生活文脈）を共有しているブランド同士か
- どのブランドペアの相関が最も強いか

| 項目 | 内容 |
|------|------|
| **回答** | 「だし連合」と「味の素ハブ」の二重構造が存在 |
| **確信度** | ★★★ 高 |
| **根拠** | GT: ほんだし×コンソメ r=0.38、SNS: 味の素が共起TOP3すべてに関与 |
| **詳細** | `output/total/report_detail/report_detail.md` |

**Q1-2. 負の相関・共倒れ**
> あるブランドが伸び悩むと、連動して伸び悩むブランドはあるか？

- ブランドAの不調がブランドBに波及するケースはあるか
- ネガティブ認識（「体に悪い」等）は特定ブランドから全体に波及するか

| 項目 | 内容 |
|------|------|
| **回答** | MSG批判は味の素→アジシオに確実に波及する |
| **確信度** | ★★★ 高 |
| **根拠** | GT: r=0.35、SNS: 両ブランドとも同率4.2%ネガティブ |
| **詳細** | `output/total/report_detail/report_detail.md` |

**Q1-3. カニバリゼーション**
> 同じCEPを取り合っているブランドはあるか？

- ブランドA↑ → ブランドB↓ の関係性はあるか
- あるとすれば、それは避けるべきか、許容すべきか

| 項目 | 内容 |
|------|------|
| **回答** | カニバリは発生していない。棲み分けが機能している |
| **確信度** | ★★★ 高 |
| **根拠** | GT: ほんだし×クックドゥ r=-0.11（棲み分け）、SNS: 共起3件のみ |
| **詳細** | `output/total/report_detail/report_detail.md` |

#### 発見

- ブランド間の相関マトリクス → ✅ 作成済み
- 「連動するブランド群」の特定 → ✅ だし連合、中華連合、うま味連合
- 施策の波及効果の可視化 → ✅ 二重構造モデルで可視化

### Issue 2: 全体最適のドライバー

#### 問いの背景

個別ブランドの最適化と、グループ全体の最適化は必ずしも一致しない。あるブランドへの投資が全体に波及効果をもたらすなら、そのブランドが「ドライバー」となる。ドライバーを特定し、そこに集中投資することで、グループ全体のブランド価値を効率的に高められる可能性がある。

#### 具体的な問い

**Q2-1. ドライバーブランドの特定**
> 味の素グループ全体にとって、最もインパクトのあるブランドはどれか？

- 投資対効果が全体に波及するブランドはどれか
- 「このブランドが伸びれば、全体が伸びる」というキーブランドはあるか

| 項目 | 内容 |
|------|------|
| **回答** | 味の素がドライバー。ただしリスクも最大 |
| **確信度** | ★★☆ 中（データ間で相違あり、条件付き） |
| **根拠** | SNS: 言及44%でハブ、GT: 検索連動は限定的。MSG批判リスク4.2% |
| **詳細** | `output/total/report_detail/report_detail.md` |

**Q2-2. 最適な訴求軸**
> ドライバーブランドで、どのような訴求をすべきか？

- 単体ブランドの最適解と、全体最適の訴求は異なるか
- どのCEP（生活文脈）を訴求すれば全体に波及するか

| 項目 | 内容 |
|------|------|
| **回答** | 「だし」が最適な訴求軸。冬場（11-12月）に集中投資 |
| **確信度** | ★★★ 高 |
| **根拠** | GT: r=0.38、季節性一致。SNS: コンソメ0.2%でネガティブリスク低 |
| **詳細** | `output/total/report_detail/report_detail.md` |

**Q2-3. 投資配分の最適化**
> 限られたリソースをどう配分すべきか？

- 全ブランドに均等配分 vs ドライバーに集中投資
- 短期的な売上 vs 長期的なブランド価値向上

| 項目 | 内容 |
|------|------|
| **回答** | 「守り」と「攻め」の二軸配分 |
| **確信度** | ★★★ 高 |
| **投資配分** | 守り40%（MSG対応30%、アジシオ連動10%）、攻め45%（だし連合30%、中華連合15%）、育成15%（香味ペースト） |
| **詳細** | `output/total/report_detail/report_detail.md` |

#### 発見

- ドライバーブランドの特定 → ✅ 味の素（条件付き）
- 全体最適の訴求軸・CEP → ✅ 「だし」、冬場集中
- 投資配分の推奨案 → ✅ 守り×攻め二軸

## フェーズ2: SNS→検索の因果検証（完了）

### Issue 3: SNSと指名検索数の相関

#### 問いの背景

フェーズ1では検索トレンド（Google Trends）とSNS言及を**別々に**分析した。しかし「SNSで話題になると検索が増えるか？」という**因果関係**は未検証。SNS投資の効果を測る指標として、SNS→検索の連動性を明らかにしたい。

#### 具体的な問い

**Q3-1. SNS言及→検索の因果**
> SNSで言及が増えると、その後検索も増えるか？

- SNS言及数↑ → 指名検索↑ の因果関係は存在するか
- SNS言及から検索までのタイムラグはどれくらいか
- ブランドによって因果の強さに差はあるか
- ポジティブ言及とネガティブ言及で効果は異なるか

| 項目 | 内容 |
|------|------|
| **回答** | **同時相関あり（コンソメで有意）、因果性は検出されず** |
| **確信度** | ★★☆ 中（データ期間35週のため） |
| **根拠** | コンソメ: r=0.460, p=0.0054（同時相関で有意）。グレンジャー因果性: 全ブランドで非有意 |
| **詳細** | `research/analysis/issue3/correlation_results.md` |

#### 分析結果サマリー

**同時相関（SNS言及数 × 検索スコア）**

| ブランド | 相関係数 | p値 | 有意 |
|---------|---------|-----|------|
| **コンソメ** | **0.460** | **0.0054** | **✓** |
| アジシオ | 0.334 | 0.0502 | - |
| 香味ペースト | 0.308 | 0.0719 | - |
| 味の素 | 0.263 | 0.1262 | - |
| ほんだし | 0.109 | 0.5313 | - |

**グレンジャー因果性検定**: 全ブランドで非有意（SNS→検索の因果は検出されず）

#### 発見

- **コンソメでSNS×検索の有意な同時相関** → SNSで話題の週は検索も増える傾向
- **因果性は未検出** → SNSが検索に「先行」する証拠はない
- **解釈**: SNSと検索は「同じ外部要因」（季節、メディア露出等）に反応している可能性

#### 施策への示唆

| 発見 | 示唆 |
|------|------|
| コンソメで同時相関 | コンソメはSNS施策の効果測定がしやすい |
| 因果性なし | SNS投資→検索増の直接効果は限定的 |
| 同時発生 | メディア露出等の外部要因が両方に影響
