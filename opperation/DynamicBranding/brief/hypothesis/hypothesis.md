## H1: 正の相関（検証済み）

### H1-1. ほんだし × クックドゥ ❌ 棄却

| 項目 | 当初仮説 | 実測結果 |
|------|----------|----------|
| **予想** | 正の相関（和食CEP共有） | r = **-0.11**（負の相関） |
| **判定** | **棄却** | |
| **解釈** | 「和食を作りたい」CEPは共有していない。むしろ**代替関係**の可能性 |
| **示唆** | ほんだし＝「自分で味付け派」、クックドゥ＝「時短派」で**セグメントが分離** |

### H1-2. 丸鶏がらスープ × クックドゥ △ 弱い支持

| 項目 | 当初仮説 | 実測結果 |
|------|----------|----------|
| **予想** | 正の相関（中華CEP共有） | r = **0.11**（弱い正の相関） |
| **判定** | **弱い支持** | |
| **解釈** | 中華料理を作る人は両方使う傾向があるが、相関は弱い |

### H1-3. コンソメ × ビストロドゥ ⚠️ 未検証

| 項目 | 内容 |
|------|------|
| **予想** | 正の相関（洋食CEP共有） |
| **状況** | ビストロドゥのデータ取得できず |
| **次のアクション** | SNS/UGCデータで検証 |

### H1-4. 味の素ハブ仮説 △ 部分的支持

| 項目 | 当初仮説 | 実測結果 |
|------|----------|----------|
| **予想** | 全ブランドと正の相関 | 平均 r = **0.11** |
| **判定** | **部分的支持** | |
| **詳細** | アジシオ(0.35)、クックドゥ(0.26)、丸鶏がら(0.20)とは正の相関。ほんだし(-0.03)、香味ペースト(-0.17)とは相関なし/負 |
| **示唆** | 味の素は「ドライバー」ではなく「中華系ブランドのハブ」 |

### H1-5. ほんだし × コンソメ ✅ 新発見

| 項目 | 内容 |
|------|------|
| **実測値** | r = **0.38**（最も強い正の相関） |
| **解釈** | 料理ジャンル（和食/洋食）ではなく**「だし」カテゴリ**として連動 |
| **共通点** | 冬型の季節性（11-12月ピーク） |
| **戦略示唆** | **「だし連合」訴求**が有効。「だしで料理がおいしくなる」統一メッセージ |

### H1-6. 味の素 × アジシオ ✅ 支持

| 項目 | 内容 |
|------|------|
| **実測値** | r = **0.35**（強い正の相関） |
| **解釈** | 「うま味」コンセプトで強く連動 |
| **リスク** | ネガティブ（MSG批判）も連動する可能性 → H3-2を支持 |

## H2: カニバリゼーション（検証済み）

### H2-1. 丸鶏がらスープ vs 香味ペースト △ 軽微

| 項目 | 当初仮説 | 実測結果 |
|------|----------|----------|
| **予想** | 負の相関（チャーハン用途で競合） | r = **-0.04** |
| **判定** | **軽微なカニバリ** | |
| **解釈** | 競合はあるが深刻ではない。用途が若干異なる（顆粒＝スープ、ペースト＝炒め物） |

### H2-2. 香味ペースト vs クックドゥ △ カニバリなし

| 項目 | 当初仮説 | 実測結果 |
|------|----------|----------|
| **予想** | 負の相関（時短中華で競合） | r = **0.04** |
| **判定** | **カニバリなし** | |
| **解釈** | 競合していない。ユーザー層が異なる |

### H2-3. ほんだし vs クックドゥ ✅ カニバリあり

| 項目 | 当初仮説 | 実測結果 |
|------|----------|----------|
| **予想** | 軽いカニバリ | r = **-0.11** |
| **判定** | **カニバリあり** | |
| **解釈** | 和食調理で代替関係。クックドゥ（時短）がほんだし（基本調味）の出番を減らしている可能性 |
| **戦略示唆** | 棲み分け訴求が必要。ほんだし＝「こだわり派」、クックドゥ＝「時短派」 |

### H2-4. 香味ペーストの孤立 ✅ 新発見

| 項目 | 内容 |
|------|------|
| **実測値** | ほんだし(-0.15)、味の素(-0.17)、アジシオ(-0.14)と負の相関 |
| **解釈** | 他ブランドと**独立した動き**をしている |
| **仮説** | 香味ペーストは独自のユーザー層（例：「チャーハン特化層」「料理男子」）を持つ |
| **戦略示唆** | 他ブランドとの連携より**独自ポジションの強化**が有効 |

| **判定** | **高リスク** |
| **解釈** | ポジティブもネガティブも連動する可能性大 |

## 戦略示唆まとめ

### 1. 「だし連合」の構築

| 項目 | 内容 |
|------|------|
| **根拠** | ほんだし × コンソメ = r = 0.38（最強の相関） |
| **訴求案** | 「だしで料理がおいしくなる」統一メッセージ |
| **施策** | 合同キャンペーン、クロスプロモーション |
| **タイミング** | 冬場（11-12月）に集中 |

### 2. 味の素の役割再定義

| 項目 | 内容 |
|------|------|
| **発見** | 全体ハブではなく「中華系ブランドのハブ」 |
| **連動ブランド** | クックドゥ(0.26)、丸鶏がら(0.20)、アジシオ(0.35) |
| **非連動** | ほんだし(-0.03)、香味ペースト(-0.17) |
| **施策** | 中華料理文脈での味の素訴求を強化 |

### 3. 香味ペーストの独自戦略

| 項目 | 内容 |
|------|------|
| **発見** | 他ブランドと連動しない独自の動き |
| **仮説** | 独自ユーザー層（チャーハン特化、料理男子等） |
| **施策** | グループ連携より**独自ポジションの強化** |
| **訴求案** | 「香味ペーストがあれば何でも作れる」 |

### 4. ほんだし × クックドゥの棲み分け

| 項目 | 内容 |
|------|------|
| **発見** | r = -0.11（カニバリ関係） |
| **解釈** | 同じ「和食」でもセグメントが分離 |
| **施策** | 明確な棲み分け訴求 |
| **ほんだし** | 「こだわり派」「基本の味」「毎日の味噌汁」 |
| **クックドゥ** | 「時短派」「特別なおかず」「週末の一品」 |
