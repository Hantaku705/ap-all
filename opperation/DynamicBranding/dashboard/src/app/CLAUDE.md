# app/

## 概要

Next.js App Routerディレクトリ。ページとAPI Routesを管理。

## ファイル一覧

| ファイル | 説明 |
|---------|------|
| `layout.tsx` | ルートレイアウト（メタデータ・グローバルスタイル） |
| `page.tsx` | トップページ（9タブダッシュボードUI） |

## ディレクトリ構成

| ディレクトリ | 説明 |
|-------------|------|
| `api/` | API Routes |
| `brands/` | ブランド詳細ページ |
| `corporate/` | コーポレート分析ダッシュボード |

## タブ構成（page.tsx）

1. トレンド推移（Google Trends / SNS言及数）
2. 相関分析
3. 季節性
4. SNS分析
5. CEP分析
6. 関連KW
7. データ
8. インサイト
9. レポート

## 関連ディレクトリ

- `../` - src/ルート
- `../components/` - UIコンポーネント

## 注意事項

- `page.tsx` は `"use client"` ディレクティブ使用（タブ切り替え等のインタラクション）
- 新規タブ追加時はタブ定義配列を更新
