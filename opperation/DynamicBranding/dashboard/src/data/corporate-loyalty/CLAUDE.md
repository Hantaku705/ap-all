# corporate-loyalty/ - ロイヤリティ別顧客インサイトデータ

## 概要

コーポレートダッシュボードの「ロイヤリティ別顧客インサイト」セクション用の静的JSONデータ。

## ファイル一覧

| ファイル | 説明 |
|---------|------|
| `corp-1-summary.json` | 味の素株式会社（corpId=1）のロイヤリティ別ペルソナデータ |

## データ構造

### ペルソナ一覧

| ロイヤリティ | ペルソナ数 | ペルソナ |
|------------|----------|---------|
| 高（27%） | 5 | 長期株式投資家、ホワイト企業志望学生、サステナ共感層、アスリートファン、タレント・CM応援層 |
| 中（68.8%） | 5 | 転職検討リサーチャー、業界ウォッチャー、個人投資家（様子見）、エンタメ視聴層、就活情報収集層 |
| 低（4.2%） | 3 | 失望した株主、添加物懸念層、インフルエンサー懐疑層 |

### ペルソナ属性

| 属性 | 説明 |
|------|------|
| `id` | 一意ID（例: high_p1） |
| `personaName` | ペルソナ名（例: 長期株式投資家） |
| `ageRange` | 年齢層（例: 40-50代） |
| `lifeStage` | ライフステージ（例: 投資家・資産運用者） |
| `interests` | 関心事リスト |
| `motivations` | 動機リスト |
| `voiceTone` | 声のトーンリスト |
| `representativeQuote` | 代表的な引用 |
| `postCount` | 推定投稿数 |
| `percentage` | このレベル内の割合 |
