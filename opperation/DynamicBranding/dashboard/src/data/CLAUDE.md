# data/ - 静的データディレクトリ

## 概要

静的JSONデータファイルを格納するディレクトリ。APIから直接読み込んで使用。

## サブフォルダ一覧

| フォルダ | 説明 |
|---------|------|
| `corporate-strategy/` | コーポレート戦略提案データ（静的） |
| `corporate-loyalty/` | **ロイヤリティ別顧客インサイトデータ（8ペルソナ）（NEW）** |
| `ws-detail/` | W's詳細分析データ（ブランド別静的JSON） |

## 関連ディレクトリ

- `../app/api/` - データを読み込むAPI Routes
- `../components/` - データを表示するコンポーネント
