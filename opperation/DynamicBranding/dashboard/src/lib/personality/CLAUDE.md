# personality/

## 概要

ブランドパーソナリティ5軸スコア算出モジュール。UGC投稿からキーワード頻度を分析し、パーソナリティスコアを生成。

## ファイル一覧

| ファイル | 説明 |
|---------|------|
| `keyword-dictionary.ts` | 5軸別キーワード辞書（重み付き） |
| `score-calculator.ts` | スコア算出ロジック（0-100正規化） |
| `personality-selector.ts` | パーソナリティタイプ判定 |
| `index.ts` | バレルエクスポート |

## 5軸定義

| 軸 | 説明 | キーワード例 |
|----|------|-------------|
| `intellect` | 知性・専門性 | プロ、本格、こだわり |
| `innovation` | 革新性 | 新しい、斬新、アレンジ |
| `warmth` | 温かみ・親しみ | 家庭、ほっこり、懐かしい |
| `reliability` | 信頼性・安定 | 定番、安心、いつもの |
| `boldness` | 大胆さ・挑戦 | 挑戦、冒険、やばい |

## 関連ディレクトリ

- `../../app/api/corporate/[corpId]/mvv/` - MVV生成API（スコア使用）
- `../../components/corporate/PersonalityRadar.tsx` - レーダーチャート表示

## 注意事項

- キーワードは日本語（ひらがな/カタカナ/漢字混在）
- 重みは0.5〜2.0の範囲で設定
- スコアは相対値（全投稿に対する比率）
