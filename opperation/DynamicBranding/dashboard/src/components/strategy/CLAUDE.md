# strategy/

## 概要

戦略分析ビューコンポーネント。ポートフォリオ全体とブランド別の戦略示唆を表示。

## ファイル一覧

| ファイル | 説明 |
|---------|------|
| `StrategyView.tsx` | 戦略分析ビュー（portfolio/brand切り替え） |

## 表示モード

| モード | 説明 |
|--------|------|
| `portfolio` | 全ブランドポートフォリオ戦略 |
| `brand` | 個別ブランド戦略（ブランドセレクタで選択） |

## セクション構成

| セクション | 内容 |
|-----------|------|
| エグゼクティブサマリー | 1-2文の概要 |
| 発見事項 | 分析から得られた事実 |
| 深掘りインサイト | 詳細な洞察 |
| 勝ちパターン | 成功事例・パターン |
| 改善機会 | 伸びしろのある領域 |
| 推奨アクション | 具体的な施策提案 |
| 競合戦略 | 競合ブランドとの比較 |

## 関連ディレクトリ

- `../` - components/ルート
- `../../app/page.tsx` - メインページ（Analyticsタブ）
- `../../../output/reports/` - レポートJSONソース

## 注意事項

- `"use client"` ディレクティブが必要
- lucide-reactアイコン多数使用
- BRANDS定数からブランドリスト取得
