# 丸鶏がらスープ ブランド分析レポート

## 戦略サマリー

**キーインサイト**: 「丸鶏がらスープ」は、時短・健康ニーズを持つ「single」層に、簡便性と健康価値を訴求することで、競合との差別化を図り、市場での存在感を高めることができる。

**エグゼクティブサマリー**: 現在の「丸鶏がらスープ」は、主要ユーザー層の不明確さや、競合製品との差別化不足が課題です。SNSデータ分析の結果、「single」層の平日夜の時短ニーズと健康志向が強く、簡便性と健康価値を両立したレシピコンテンツが有効であることが判明しました。今後は、「single」層に特化した、時短かつ健康的なレシピ提案を強化し、UGCを積極的に活用したコミュニケーション戦略を展開することで、ブランドロイヤリティの向上と市場シェアの拡大を目指します。特に動画コンテンツの拡充はエンゲージメント向上に繋がる可能性が高いです。

### 主な発見事項
- 主要ユーザー層が「unknown」で81%を占めており、ターゲット層の明確化が急務である。
- 「平日夜の時短ニーズ」が最も高い利用文脈であり、簡便レシピへの強いニーズが確認できる。
- 「single」層は「一人暮らしの手抜き飯」への親和性が高く、この層へのアプローチが有効。
- 競合製品（味の素、ほんだし等）との同時言及が多く、独自性の確立が求められる。
- エンゲージメント上位投稿は具体的なレシピ提案が多く、ユーザーは実用的な情報を重視する。

### 推奨アクション
- ターゲット層を「single」層に明確化し、パーソナライズされたコンテンツを提供する。
- 「平日夜の時短」と「健康志向」を両立した、簡便レシピの動画コンテンツを拡充する。
- UGCを活用し、ユーザー参加型のキャンペーンやコミュニティを形成する。
- 健康価値（アミノ酸、ミネラル等）を強調し、競合製品との差別化を図る。
- ネガティブ投稿を分析し、「味が濃い」「添加物が気になる」などの懸念点に対応する。

## ユーザー像

*誰が「丸鶏がらスープ」を使っているか？*

### 発見事項
- ライフステージ不明が81%と高いが、シングル層の投稿も約5%（1,539件中77件）見られる(49233, 47166, 57846, 41872)。
- 調理対象は「自分向け」が58%だが、「家族向け」も20%存在し、子育て層からの「息子に美味しい牛タンの見分け方を教えた」という声も(47663, 1106, 15)。
- ポジティブな投稿が約70%を占める一方で、約20%は「舌に残る感じが苦手」といった味覚や「質量の半分が塩分」という成分への不満も(32261, 47988, 47166, 60242)。
- レシピ共有では「鶏むね肉」の活用に関心が高く、「鶏むね肉のしっとり蒸し鶏」といった具体的なレシピ名も言及されている(70808, 3751)。
- 簡便性を求める声として、「麺つゆ飽きたから、鶏ガラスープ…ぶち込んでうどんの汁作ったら」というように、複数の調味料を組み合わせる例がある(47166)。

### データ
| 属性 | 件数 | シェア |
| --- | --- | --- |
| ライフステージ: unknown | 804 | 81% |
| ライフステージ: single | 125 | 13% |
| ライフステージ: child_raising | 50 | 5% |

### インサイト
- シングル層は、手軽に済ませられる簡便な味付けを求めている傾向があり、複数の調味料を組み合わせて独自の味を追求する様子が見られる。
- 家族向けでは、焼肉の下味に利用するなど汎用的な利用方法が見られる一方、味覚や塩分に関するネガティブな意見も一定数存在する。
- 鶏むね肉レシピへの関心の高さから、ヘルシー志向のユーザーに訴求できる可能性がある。

### 推奨アクション
- マーケティングチームは、シングル層向けに「#俺の鶏ガラ飯」のような手軽なアレンジレシピ投稿を促すキャンペーンを、3ヶ月以内にSNSで実施する。
- 商品開発チームは、既存の鶏むね肉レシピ投稿（70808, 3751）を参考に、ヘルシー志向ユーザー向けの鶏がらスープを使ったレシピを開発し、1ヶ月以内に公開する。
- カスタマーサポートチームは、塩分に関する懸念に対し、減塩レシピや少量使用での活用法をまとめたFAQページを1週間以内に公開する。

## DPT分析（Use Case×Positioning）

*どのような場面で、何が差別化価値となっているか？*

### 発見事項
- ポジティブなUGC投稿が約49%（1539件中）を占め、利用シーンとして「家族向け料理」と「一人暮らし向け料理」が確認できる(ID:3952, ID:49233)。
- ネガティブな投稿は約3%（1539件中）で、「味が苦手」「入れ間違い」など、風味や使用時の誤りに関する意見が見られる(ID:32261, ID:29136)。
- 鶏がらスープの素は、スープ以外にも「チャーハン」「焼肉」「パスタ」など多様な料理で利用されていることが確認できる(ID:49233, ID:47663, ID:3078)。
- 「#丸鶏がらスープ」のように特定の製品名がハッシュタグとして使用されているUGCが確認でき、レシピ共有の文脈で活用されている(ID:3078)。
- ライフステージが判明しているUGCのうち、singleの割合がchild_raisingより高い。一方で、family向けの投稿も一定数存在する(ID:49233, ID:47663)

### インサイト
- 鶏がらスープの素は汎用性が高く、様々な料理で活用されているが、一部ユーザーは風味に不満を感じているため、風味に関するポジ/ネガ両面のレビューを分析する必要がある。
- 簡便性を重視するユーザーが多いと考えられるため、ライフステージごとのニーズ（時短、健康志向など）に合わせたレシピ提案が有効である。

### 推奨アクション
- 製品開発チームは、風味に関するネガティブな意見（約3%）を詳細に分析し、風味改善の検討を2週間以内に開始する（レビュー抽出と分類）。
- マーケティングチームは、シングル層向けには「#簡単レシピ」を活用した簡便なレシピ、家族向けには栄養バランスを考慮したレシピを、1ヶ月以内にSNSで展開する（レシピ作成と投稿）。
- 広報部は、UGCでポジティブな評価を得ている「#丸鶏がらスープ」を活用したレシピ投稿を促進するため、インフルエンサーとの連携を1ヶ月以内に開始する(インフルエンサー選定と交渉)。
