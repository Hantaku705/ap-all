# reports/

## 概要

ブランド別レポートのJSON/Markdownファイル。LLMインサイト付きの動的レポート。

## ディレクトリ構成

| パス | 説明 |
|------|------|
| `index.json` | レポート一覧 |
| `portfolio/` | ポートフォリオ全体レポート |
| `{ブランド名}/` | ブランド別レポート（8フォルダ） |

## ブランドレポート構成

| ファイル | 説明 |
|---------|------|
| `report.json` | 構造化レポートデータ |
| `report.json.bak` | バックアップ（品質改善前） |
| `report.md` | Markdown形式レポート |

## 対象ブランド

ほんだし、クックドゥ、味の素、丸鶏がらスープ、香味ペースト、コンソメ、ピュアセレクト、アジシオ

## 関連ディレクトリ

- `../../src/lib/report-quality/` - 品質改善モジュール
- `../../src/app/api/reports/` - レポートAPI

## 注意事項

- `.bak`ファイルは品質改善前のバックアップ
- Markdownはダウンロード用
