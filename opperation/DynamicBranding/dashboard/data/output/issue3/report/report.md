# Issue 3 分析レポート: SNSと指名検索数の相関

**分析日**: 2026-01-16
**分析期間**: 2025年3月〜10月（35週間）
**SNS言及総数**: 1,092件

---

## エグゼクティブサマリー

### 結論

| 問い | 回答 |
|------|------|
| **SNS→検索の因果あり？** | **同時相関はあるが、因果性は未検出** |
| **最も相関が高いブランド** | コンソメ（r=0.460, p<0.01） |
| **因果性が確認されたブランド** | なし（0/8ブランド） |

### 主要な発見

1. **コンソメでSNS×検索の有意な同時相関を確認**
   - 相関係数 r=0.460（中程度の正の相関）
   - p値=0.0054（統計的に有意）

2. **因果性（SNS先行→検索追随）は検出されず**
   - グレンジャー因果性検定: 全ブランドで非有意
   - 解釈: SNSと検索は「同じ外部要因」に反応している可能性

3. **ブランドによって相関の強さが異なる**
   - 高相関: コンソメ、アジシオ、香味ペースト
   - 低相関: ほんだし、クックドゥ、丸鶏がらスープ

## 1. 分析手法

### データソース

| データ | 期間 | 粒度 | 件数 |
|--------|------|------|------|
| SNS言及 | 2025年3月〜10月 | 日次→週次集計 | 1,092件 |
| Google Trends | 同上 | 週次 | 35週 |

### 分析手法

1. **同時相関**: SNS言及数 × 検索スコア（同週）
2. **ラグ付き相関**: SNS言及数(t-n週) × 検索スコア(t週)、n=1〜4
3. **グレンジャー因果性検定**: 過去のSNS言及が現在の検索を予測するか

## 5. 考察

### なぜ因果性が検出されなかったか

1. **同時発生**: SNSと検索が「同じ外部要因」に反応している可能性
   - 季節要因（冬場の鍋需要など）
   - メディア露出（テレビCM、ニュース）
   - キャンペーン

2. **データ期間の制約**: 35週間では因果性の検出が困難
   - より長期のデータで再検証が必要

3. **SNS→検索の経路が複雑**
   - SNS → 認知 → 検討 → 検索 のタイムラグが4週以上の可能性

### コンソメの同時相関が高い理由

- **レシピ文脈**: SNSでレシピが拡散 → 同時に検索も増加
- **季節性が明確**: 冬場の鍋・スープ需要と連動
- **メディア露出との連動**: CM放映時にSNS言及と検索が同時増加

## 6. 施策への示唆

| 発見 | 施策への示唆 |
|------|-------------|
| コンソメで同時相関 | SNS施策の効果測定指標として「検索数」が使える |
| 因果性なし | SNS投資→検索増の直接効果は限定的 |
| 同時発生 | メディアミックス（TV+SNS）で相乗効果を狙う |
| ブランド差 | コンソメ・アジシオはSNS施策の検証がしやすい |

### 推奨アクション

1. **コンソメのSNS施策を強化**
   - SNSと検索の連動が確認されたため、効果測定が可能
   - 冬場の鍋・スープレシピ投稿を増やす

2. **メディアミックス施策の実施**
   - TV CM と SNS を同時に展開
   - 同じ外部要因として両方に効く可能性

3. **より長期のデータ収集**
   - 1年以上のデータで因果性を再検証
   - 季節性の影響を正確に分離

## 7. 分析の限界

| 限界 | 影響 |
|------|------|
| データ期間が短い（35週） | 因果性の検出が困難 |
| SNS言及数が少ないブランドあり | 相関の信頼性が低い |
| 外部要因を統制していない | 季節性・メディア露出の影響を分離できず |
