## 3. 主要な発見

### 発見1: だし連合（ほんだし × コンソメ）

| 項目 | 内容 |
|------|------|
| **相関係数** | r = 0.38（最も強い正の相関） |
| **統計的有意性** | p < 0.01 |
| **サンプル数** | n = 262週 |

**解釈**:
- 料理ジャンル（和食/洋食）を超えて「だし」として連動
- 同じ季節性パターン（冬型）
- 「だしで料理がおいしくなる」という共通認知

---

### 発見2: うま味連合（味の素 × アジシオ）

| 項目 | 内容 |
|------|------|
| **相関係数** | r = 0.35（強い正の相関） |
| **統計的有意性** | p < 0.01 |
| **サンプル数** | n = 262週 |

**解釈**:
- 「うま味」コンセプトで強く連動
- 商品名に「味の素」を含む関連性
- ポジ/ネガ両方が連動するリスク

---

### 発見3: ほんだし × クックドゥの棲み分け

| 項目 | 内容 |
|------|------|
| **相関係数** | r = -0.11（弱い負の相関） |
| **当初仮説** | 和食CEP共有で正の相関 |
| **実際** | 負の相関（棲み分け） |

**解釈**:
- 同じ「和食」でもセグメントが分離
- ほんだし＝「自分で味付け派」
- クックドゥ＝「時短派」

---

### 発見4: 香味ペーストの孤立

| 項目 | 内容 |
|------|------|
| **相関パターン** | 複数ブランドと負の相関 |
| **特徴** | 他ブランドと独立した動き |

**解釈**:
- 独自のユーザー層を持つ可能性
- チャーハン特化、料理男子など

## 4. 季節性分析

### 月別パターン

| ブランド | パターン | ピーク月 | 底月 | 変動幅 |
|---------|---------|---------|------|--------|
| **コンソメ** | **冬型** | 11月 | 7月 | **23.3** |
| ほんだし | 弱冬型 | 12月 | 6月 | 4.3 |
| 味の素 | 平坦 | 5月 | 12月 | 9.6 |
| クックドゥ | 平坦 | 3月 | 11月 | 1.9 |
| 丸鶏がら | やや冬型 | 12月 | 8月 | 3.8 |
| アジシオ | 平坦 | - | - | 低 |
| 香味ペースト | 平坦 | - | - | 低 |

### インサイト

- **コンソメの季節性が最大** - 変動幅23.3。冬場に集中投資が有効
- **ほんだし・コンソメは冬連動** - だし連合の季節性も一致
- **クックドゥは通年安定** - 季節に依存しない需要

## 5. 味の素ハブ仮説の検証

### 当初仮説

> 味の素は全ブランドと正の相関を持つ「ハブ」として機能

### 検証結果

| ペア | r値 | 評価 |
|------|-----|------|
| 味の素 × アジシオ | 0.35 | 強い |
| 味の素 × クックドゥ | 0.26 | 中程度 |
| 味の素 × 丸鶏がら | 0.20 | 中程度 |
| 味の素 × コンソメ | 0.06 | 弱い |
| 味の素 × ほんだし | -0.03 | なし |
| 味の素 × 香味ペースト | -0.17 | 負 |

**平均相関係数**: r = 0.11

### 結論

| 項目 | 内容 |
|------|------|
| **判定** | 部分的支持 |
| **修正仮説** | 「全体のハブ」ではなく「中華系ブランドのハブ」 |
| **連動ブランド** | アジシオ、クックドゥ、丸鶏がら |
| **非連動ブランド** | ほんだし、香味ペースト、コンソメ |

## 6. データの限界

| 限界 | 影響 |
|------|------|
| 検索≠購買 | 興味≠行動。購買データでの検証推奨 |
| 相関≠因果 | 第三の要因の可能性。実験的検証が必要 |
| 相対値 | 絶対的な検索ボリュームは不明 |
| 一部ブランド | 検索ボリュームが小さく不安定な可能性 |
