# by_data/ - Data別分析レポート

データソースごとの基礎分析レポートを格納。
Issue回答を含まない純粋なデータ分析。

---

## フォルダ構成

| フォルダ | データソース | 内容 |
|---------|-------------|------|
| `googletrend/` | Google Trends | 相関・季節性・時系列 |
| `sns/` | SNSデータ | 言及数・共起・センチメント |

---

## 各レポートの概要

### googletrend/report.md

| 項目 | 内容 |
|------|------|
| データ期間 | 過去5年（262週） |
| 主要な発見 | だし連合（r=0.38）、うま味連合（r=0.35） |
| 季節性 | コンソメが冬型（変動幅23.3） |

### sns/report.md

| 項目 | 内容 |
|------|------|
| データ件数 | 50,000件（content有: 8,517件） |
| 主要な発見 | 味の素が言及44%、共起の中心 |
| センチメント | 味の素・アジシオが同率4.2%ネガティブ |
