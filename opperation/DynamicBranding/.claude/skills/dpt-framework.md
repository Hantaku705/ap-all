# DPT（Dynamic Positioning Table）フレームワーク

## 概要

DPT（Dynamic Positioning Table）は、**Use Case（生活文脈）× Positioning（競合との差別化）**をセットで捉えるフレームワークです。

これは単なる整理表ではなく、Use Case（消費者が商品を手に取る生活文脈）ごとに、競合と競争要素（POP／POD）を明確化し、それぞれ異なる戦略を導き出すためのマーケティングツールです。

---

## DPTを構成する要素

### ① Use Case（生活文脈）

商品が使われる具体的な生活のシーン・文脈。

| 要素 | 説明 |
|------|------|
| **Why（理由・動機）** | 商品を手に取る具体的な動機やニーズ |
| **When／Where（シーン）** | 商品を利用する具体的な時間帯や場所 |

これらを組み合わせて **「消費者の生活におけるリアルな使用場面（Use Case）」** を特定します。

### ② Positioning（競合との差別化）

特定したUse Caseごとに、競合ブランドと比較し、自社商品の位置付けを定めます。

| 要素 | 説明 |
|------|------|
| **競合ブランド** | 消費者がそのUse Caseで比較検討する商品群 |
| **POP（Points of Parity）** | 消費者がそのUse Caseにおいて、競合商品に求める共通の価値や要素 |
| **POD（Points of Difference）** | 競合商品にない、自社だけが持つ差別化された価値や特性 |

## DPTの活用例

| Use Case（生活文脈） | Positioning（競合との差別化） |
|---------------------|------------------------------|
| **背徳夜食**<br>- Why：ガッツリ食べたい時<br>- When／Where：平日深夜、週末夜、自宅 | - 競合：カップ麺、冷凍ピザ、ファストフード<br>- POP：ジャンク感、濃厚な味わい<br>- POD：「パリッ！」ジューシー、スパイス感 |
| **ヘルシー夜食**<br>- Why：罪悪感なしで満腹感を得たい時<br>- When／Where：ダイエット中、運動後、自宅 | - 競合：大豆ミート製品、ヘルシー冷凍食品<br>- POP：ヘルシー感、手軽さ、低カロリー<br>- POD：「パリッ！」脂質少なめ、高プロテイン |
| **おつまみ**<br>- Why：料理代わりのおつまみ需要<br>- When／Where：週末夜、晩酌時間帯、自宅・友人宅 | - 競合：チーズ、生ハム、燻製デリ<br>- POP：調理不要、手軽にお酒と楽しめる味わい<br>- POD：「パリッ！」食感、リッチなフレーバー、温感 |

---

## DPTの狙い・意義

1. **Use Caseごとの戦略設計**: それぞれのUse Caseで異なるターゲットや競合を明確にし、個別具体的なマーケティング戦略を設定可能
2. **説得力ある差別化訴求**: 消費者のリアルな生活シーンを基点に、説得力ある差別化訴求を構築できる

## 実例：シャウエッセンのDPT

### Use Case ①：お手軽時短ごはん

| 項目 | 内容 |
|------|------|
| **Why（理由）** | 短時間で簡単に満足感のある料理を作りたい時 |
| **When／Where（シーン）** | 平日の朝や夜、自宅のキッチン |
| **競合ブランド** | 冷凍食品、レトルト食品、インスタント食品 |
| **POP（類似要素）** | 調理が簡単、時短調理、安定した味付け |
| **POD（差別化要素）** | 「パリッ！」とした食感、ボイルするだけで専門店クオリティのジューシーさ |

### Use Case ②：家族みんなが喜ぶ食卓の定番

| 項目 | 内容 |
|------|------|
| **Why（理由）** | 家族みんなで楽しく、美味しいものを共有したい時 |
| **When／Where（シーン）** | 休日の昼食、夕食、自宅の食卓 |
| **競合ブランド** | 他のメーカーのウインナー、ハンバーグ、から揚げなどの定番おかず |
| **POP（類似要素）** | 子供から大人まで楽しめる味、調理が簡単 |
| **POD（差別化要素）** | 「シャウエッセン以外許せない」というファン心理を形成するほどの圧倒的な味の差、ブランドへの安心感 |

### Use Case ③：夜のお酒のお供

| 項目 | 内容 |
|------|------|
| **Why（理由）** | 料理代わりに手軽にお酒を楽しみたい時 |
| **When／Where（シーン）** | 平日・休日の夜、自宅や友人宅での晩酌 |
| **競合ブランド** | チーズ、生ハム、燻製デリ、おつまみ用スナック |
| **POP（類似要素）** | お酒との相性、手軽さ、調理不要な気軽さ |
| **POD（差別化要素）** | ブラックペッパーが効いた「夜味」など、スパイス感が強く差別化されたバリエーションの存在、「パリッ！」食感による満足感の高さ |

### Use Case ④：特別感のある日常の贅沢

| 項目 | 内容 |
|------|------|
| **Why（理由）** | 日常の食事にちょっとした贅沢感を加えたい時 |
| **When／Where（シーン）** | 特別ではない日常、自宅でリラックスしている時 |
| **競合ブランド** | プレミアムウインナーやソーセージブランド（アルトバイエルンなど） |
| **POP（類似要素）** | 高品質感、ジューシーさ、豊かな風味 |
| **POD（差別化要素）** | 「シャウエッセン」の圧倒的なブランド認知度と、「パリッ！」という他にはない食感、マヨネーズやタバスコとの抜群の相性など味付けアレンジの楽しさ |

## 活用方法

1. **口コミ・UGCからUse Caseを抽出**: SNS投稿や口コミからリアルな利用シーンを特定
2. **各Use Caseで競合を特定**: そのシーンで消費者が比較検討する商品群を明確化
3. **POP/PODを整理**: 共通価値と差別化要素を言語化
4. **マーケティング施策に展開**: Use Caseごとに最適なコミュニケーション戦略を設計
