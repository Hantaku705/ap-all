# /resume - セッション再開時の読み込み

前回のセッションから作業を再開します。

## 実行手順

1. HANDOFF.md を読み込む
2. 内容をユーザーに報告:
   - 前回の完了/作業中タスク
   - 次のアクション（優先順位付き）
   - 未解決の問題
   - 未コミット変更の有無
3. git status で現在の状態を確認
4. 「何から始めますか？」と確認

## 出力フォーマット

```
## 前回の状態

### 完了済み
- [タスク一覧]

### 作業中
- [タスク一覧]

### 次のアクション（優先順）
1. [アクション]
2. [アクション]

### 未解決の問題
- [問題]

---

**今回は何から始めますか？**
```

## 注意事項

- HANDOFF.md が存在しない場合は新規セッションとして案内
- 未コミット変更がある場合は先に確認を促す
- git stash がある場合は stash pop を提案
